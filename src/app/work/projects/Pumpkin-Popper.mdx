## Overview

This project involved creating a fun and engaging arcade shooting game designed for various devices. The focus was on developing responsive controls and satisfying gameplay mechanics to provide an enjoyable arcade experience.

## Key Features

- **Engaging Gameplay**: Designed core mechanics for a fast-paced and enjoyable shooting experience.
- **Responsive Controls**: Implemented intuitive and highly responsive controls optimized for device interaction.
- **Level Design**: Created levels with increasing difficulty and varied challenges.

## Technologies Used

- **Unity3D Engine & C#**: Developed the game logic, mechanics, and level structure.

## Challenges and Learnings

Balancing gameplay difficulty and ensuring smooth performance across different devices were key challenges. Optimizing assets and scripts for a consistent arcade experience provided valuable learning in mobile/device game development.

## Outcome

The resulting arcade shooting game offered a fun and accessible experience for players on various devices, demonstrating proficiency in core game development principles.
